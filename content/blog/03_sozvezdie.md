## Что за проект

Сайт АО "Концерн "Созвездие" — представляет собой официальный ресурс АО «Концерн «Созвездие» — ведущего российского предприятия радиоэлектронного комплекса.

## Назначение сайта

Сайт служит корпоративной платформой для представления продуктов, услуг и достижений компании: от профессиональных систем радиосвязи (включая линейку DMR) до решений промышленного интернета вещей и телекоммуникационного оборудования. Он содержит разделы о истории, контактах, вакансиях и продукции, подчёркивая надёжность и инновационность решений, часто превосходящих западные аналоги.

## Моё участие в проекте

Моё участие в проекте по разработке сайта АО «Концерн «Созвездие» включало полную реализацию интерфейса по предоставленным Figma-макетам с обеспечением адаптивной верстки для всех типов устройств. 
В рамках работ были разработаны и внедрены ключевые страницы сайта, а также интеграция форм обратной связи для оперативной обработки запросов пользователей.

## Технический стек

* Фреймворк: Vue 3
* Инструмент сборки: Vite 2, ESBuild
* Менеджер пакетов: pnpm
* Язык: TypeScript
* Стилизация: Windi CSS (Tailwind CSS по требованию)
* Маршрутизация: Vue Router с файловой маршрутизацией (`vite-plugin-pages`) и макетами

## Заключение

Работа над этим проектом дала опыт полноценной реализации корпоративного сайта «под ключ» на современном стеке Vue 3 + Vite + TypeScript с использованием утилитарной стилизации.
Получил опыт разработки официального сайта крупного предприятия радиоэлектронного комплекса, где важны имидж, строгий стиль и соответствие бренд-гайдам.
